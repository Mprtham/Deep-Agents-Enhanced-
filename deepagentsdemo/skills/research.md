# Skill: Deep Research

## Purpose
Conduct comprehensive research on any topic using web search.

## Steps
1. Break the topic into 3-5 sub-questions
2. Search each sub-question individually
3. Cross-reference results across sources
4. Synthesise into a structured summary with citations

## Output Format
- Executive Summary (2-3 sentences)
- Key Findings (bullet list)
- Sources (URLs)
- Confidence Level (High / Medium / Low)
