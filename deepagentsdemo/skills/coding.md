# Skill: Code Generation

## Purpose
Write clean, production-quality code in any language.

## Guidelines
- Understand requirements fully before writing
- Use idiomatic patterns for the target language
- Include brief inline comments only where the WHY is non-obvious
- Write tests alongside implementation when asked
- Prefer clarity over cleverness

## Output Format
- Language clearly stated
- Code block with syntax highlighting
- Brief explanation of key design choices
